# Changelog

All notable changes to this project are documented in this file.

## [v1.0.0] - 2026-06-10

### Added

* Initial Shorts removal.
* Strict Search Title Match.
* Popup controls.
* Theme Color Test.
* Theme Surface Test.
* Hide views and likes for visible view-count and like-count text.
* Icon support.
* Public release packaging.

### Fixed

* Hide views and likes preserves Like and Dislike controls while hiding only count text.
* Mutation handling debounces added-node scans for better YouTube page responsiveness.
* Content-script Chrome API calls stop cleanly if an old extension context is invalidated.
