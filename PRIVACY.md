# Privacy Policy

ShortsNuker does not collect personal data.

ShortsNuker does not transmit data to external servers.

ShortsNuker does not use analytics or telemetry.

ShortsNuker stores only local extension settings and toggle states through Chrome extension storage.

ShortsNuker runs only on YouTube pages matched by `*://*.youtube.com/*`.

ShortsNuker does not sell data.

ShortsNuker does not track browsing history.

ShortsNuker does not require a user account.

ShortsNuker does not make network calls, perform remote logging, or connect to any external service.
